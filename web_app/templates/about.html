Gesture Control System

Control Your Computer With Hand Gestures

What is Gesture Control?

Gesture Control System is an advanced hand gesture recognition application that allows you to control your computer using natural hand movements. Powered by MediaPipe and OpenCV, it provides real-time gesture detection with amazing visual effects and customizable commands.

Key Features

5 Visual Effects
  • 🌈 Rainbow Cursor
  • ✨ Particle Trail with Gravity
  • 💎 Hologram Effect
  • 🧲 Magnetic Field
  • 🎨 Motion Trail
Advanced Features
  • 🔷 Shape Detection (Circle, Line, Polygon)
  • 🎙️ Gesture Recording
  • 🤖 AI-Powered Commands
  • ⚡ Real-time Speed Analysis
  • 💀 Hand Skeleton Visualization

How It Works

1
Webcam Detection

Captures your hand position in real-time using your webcam

2
Hand Recognition

Uses MediaPipe to detect all 21 hand landmarks with 99% accuracy

3
Gesture Analysis

Analyzes hand pose and movement patterns to recognize gestures

4
Command Execution

Converts recognized gestures into computer commands

Gesture Commands

Mouse Control
  • Index + Thumb: Left Click
  • Middle + Thumb: Scroll
  • Ring + Thumb: Right Click
  • Hand Position: Move Cursor
Keyboard Shortcuts
  • 1-5: Visual Effects
  • 6: Normal Mode
  • S: Shape Detection
  • R: Record Gesture
